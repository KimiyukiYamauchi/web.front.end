## ✅ Google Analytics 設定手順

### ① Google Analytics にアクセス

👉 https://analytics.google.com/

Google アカウントでログインします。

（Google アカウントがまだない場合は、先に作成しておきます）

### ② プロパティを作成（初めての場合）

- 左下の をクリック。
- 「アカウントを作成」または既存アカウントを選択。
- 「プロパティを作成」をクリック。
- ウェブサイトやアプリの情報を入力します。
- プロパティ名（例：My Website）
- レポートのタイムゾーン（例：日本）
- 通貨（例：日本円）

### ③ データストリームを作成

- プロパティ作成後、「データストリームを追加」をクリック。
- 「ウェブ（Web）」を選択。
- あなたの Web サイトの URL と名前を入力して「ストリームを作成」。

### ④ 測定 ID（Measurement ID）の確認

- ストリームが作成されると、
- 画面右上に が表示されます。
- 例： G-XXXXXXX
  この ID があなたの Web サイトに埋め込むコードで使う ID です。

### ⑤ 測定 ID の確認場所（後から見る場合）

もしあとで確認したいときは：

- 左下「管理」 → プロパティ列で対象プロパティを選択
- 「データストリーム」→ Web ストリームをクリック
- 右上に が表示されます。
